# TFG-Simulaciones-Python
Simulaciones el python para el TFG titulado "Antes del Big Bang: de la inflación al universo caliente".
## Contenidos del documento TFG_códigos.ipynb
1. Simulación de la dinámica del sistema saxión-axión.
2. Simulación de la creación de carga U(1) y su posterior conservación.
3. Construcción de la región en la que el modelo es válido
