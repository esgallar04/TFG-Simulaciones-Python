# TFG Simulaciones Python
Simulaciones el python para el TFG titulado "Antes del Big Bang: de la inflación al universo caliente". Estos códigos han sido realizados con Python con ayuda de la IA. No obstante, todos ellos han sido revisados por mí y he tenido que hacer numerosos cambios para asegurar que las gráficas resultantes fuesen las adecuadas. Además, he incluido comentarios explicativos para asegurar un rápido entendimiento de los códigos al ser leídos.
## Contenidos del documento TFG_códigos.ipynb
1. Simulación de la dinámica del sistema saxión-axión.
2. Simulación de la creación de carga U(1) y su posterior conservación.
3. Construcción de la región en la que el modelo es válido
